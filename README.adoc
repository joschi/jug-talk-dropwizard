== Build instructions

* Install dependencies:

  $ rm -f Gemfile.lock
  $ bundle --path=.bundle/gems --binstubs=.bundle/.bin install

* Build slides:

  $ make

* Open slides in browser:

  $ make open

* PDF export: https://github.com/hakimel/reveal.js/tree/3.3.0#pdf-export
